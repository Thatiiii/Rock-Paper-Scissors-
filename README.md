# Rock-Paper-Scissors-
 A simple JavaScript-based Rock Paper Scissors game developed for fun and practice. 
 This project demonstrates the use of basic JavaScript logic, and user interaction in a lightweight browser game. 

**Features**
- Classic Rock, Paper, Scissors gameplay
- Interactive with user input required
- Randomized computer opponent
- Instant win/lose/draw results displayed on screen
- Clean and beginner-friendly code structure
